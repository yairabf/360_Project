Welcome to 360 building view

Create a zip file containing the following:
  • The 360 panoramic pictures you took using the mobile app and the 360 camera.
  • The info.json file created by the mobile app.
  • The map.JPEG file created by the mobile app.

** note that the map file must be a .JPEG file and the zip file must not contain a folder that contains all those file. **

Please choose an option

Quick mode:

Creating the view from your last pictures you took in each picturing point.

Custom mode:

After uploading the panoramic pictures, you will be able to choose your pictures for each view.
